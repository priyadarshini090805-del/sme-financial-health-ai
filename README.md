# SME Financial Health AI Platform

This project is an AI-powered platform designed to assess and improve the financial health of Small and Medium Enterprises (SMEs). It analyzes financial statements, cash flows, and business metrics to provide clear, actionable insights.

## Key Capabilities

- Financial health assessment
- Cash flow analysis
- Creditworthiness evaluation
- Risk detection and alerts
- Cost optimization recommendations
- Working capital insights
- Financial forecasting
- Investor-ready financial reports
- Multilingual support for business owners

## Supported Data Sources

- CSV / XLSX financial files
- Text-based PDF exports
- Optional banking API integration
- Optional GST return data import

## Tech Stack

- Backend: Python (FastAPI)
- Data Processing: pandas
- AI Layer: OpenAI GPT-5
- Frontend: React.js
- Database: PostgreSQL
- Security: Encrypted data at rest and in transit
